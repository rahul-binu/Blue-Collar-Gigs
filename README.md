## Run Locally

Clone the project

```bash
  git clone https://github.com/rahul-binu/Blue-Collar-Gigs.git
```

Go to the project directory

```bash
  cd blue-collar-gigs
```

Install dependencies

```bash
  npm install
  npm install bootstrap --save
  npm install axios --save
```

Start the server

```bash
  npm run dev
```
